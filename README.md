# Тестирование

## Установка через Poetry фреймворк Pytest и Code Coverage
1. Введите в терминале команду:
~~~
poetry add --group dev pytest
~~~

2. Запустите тесты с помощью команды pytest.
Веедите в терминале команду:
~~~
pytest
~~~

3. В pytest для анализа покрытия кода надо поставить библиотеку 
pytest-cov:
Веедите в терминале команду:
~~~
poetry add --group dev pytest-cov
~~~

4. Чтобы запустить тесты с оценкой покрытия, можно воспользоваться следующими командами:
~~~
pytest --cov
~~~
 — при активированном виртуальном окружении.

5. В файле conftest.py находятся фикстуры - своего рода ячейки с данными, которые  
используются в файлах для тестов (файлах расположенных в папке tests с префиксом test_)

6. Чтобы сформировать отчёт о покрытии, можно ввести команду в терминале:
~~~
pytest --cov=src --cov-report=html
~~~

Отчет о покрытии в HTML-формате находится в папке htmlcov
 и храниться в файле с названием index.html.
Открывать данный отчёт необходимо с помощью браузера (Open In -> Browser).
